Converting from decimal to binary

The process for converting from decimal to binary can be done in several ways.

One algorithm is to repeatedly divide the binary value by 2 until zero is reached, keeping track of the remainder each time.

For example, converting 43 to binary leads to:

  • 43 / 2 = 21 remainder 1
  • 21 / 2 = 10 remainder 1
  • 10 / 2 = 5 remainder 0
  • 5 / 2 = 2 remainder 1
  • 2 / 2 = 1 remainder 0
  • 1 / 2 = 0 remainder 1

So reading from the bottom up, 43 in binary is 101011. We can check this by converting back, recalling that we start numbering positions at 0 from right to left: 20 + 21 + 23 + 25 = 1 + 2 + 8 + 32 = 43.
